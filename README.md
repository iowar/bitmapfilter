# Bitmap Filter

Bitmap filter implementation.

## Install

```
$ git clone https://github.com/iowar/bitmapfilter.git
$ cd /bitmapfilter
```

## Dependencies
```
$ pip install -r requirements.txt
```

## Run
```
$ python bitmap.py test.jpg
```

final state is saved as out.bmp file in the same directory.
